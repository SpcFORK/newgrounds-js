# Installation

## NPM

```
npm i newgrounds.js
```

## CDN

```js
import { newgrounds } from "https://cdn.jsdelivr.net/npm/newgrounds.js@3.1.0/dist/newgrounds.mjs";
```

## HTML Tag

index.js

```html
<script src="https://cdn.jsdelivr.net/npm/newgrounds.js@3.1.0/dist/newgrounds.js"></script>
```
